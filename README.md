# Banking Analytics Portfolio

This repository contains applied data science projects focused on
real-world banking and financial risk problems.

## Projects

### 1. Credit Risk Model
Predicting loan default probability using supervised machine learning
with a focus on interpretability and regulatory relevance.

### 2. Fraud Detection
Detecting fraudulent transactions using imbalanced classification
and precision-recall optimisation.

### 3. Economic Forecasting
Time-series forecasting of macroeconomic indicators to support
lending and capital risk decisions.

## Tools & Skills
- Python, SQL, Pandas, Scikit-learn
- Credit risk modelling
- Imbalanced classification
- Time-series forecasting
- Model interpretability
